# 📚 Salsabila Bookstore - Landing Page

Website resmi *landing page* **Salsabila Bookstore**, toko buku cerita anak bergambar premium yang berfokus pada pembangunan karakter (*character building*), empati, dan kecerdasan interpersonal anak usia dini (4–7 tahun).

Proyek ini dioptimalkan khusus untuk keperluan kampanye iklan digital seperti **Meta Ads (Facebook Ads)** dengan performa muat halaman yang cepat dan responsif di perangkat seluler (*mobile-friendly*).

---

## 🛠️ Teknologi yang Digunakan

*   **HTML5** – Struktur semantik untuk performa SEO dan aksesibilitas web yang optimal.
*   **Tailwind CSS (v4 via CDN)** – Kerangka kerja desain (*utility-first*) untuk tampilan antarmuka yang modern, dinamis, dan responsif.
*   **Google Fonts** – Menggunakan font *Plus Jakarta Sans* dan *Quicksand* untuk memberikan kesan ramah anak sekaligus profesional.

## 🚀 Fitur Utama

*   **AIDA Copywriting Framework:** Alur halaman disusun terstruktur mulai dari menarik perhatian (*Attention*) hingga tombol aksi (*Action*).
*   **Direct WhatsApp Integration:** Setiap tombol pemesanan produk dilengkapi dengan parameter teks otomatis langsung menuju WhatsApp Admin.
*   **Fully Responsive:** Desain otomatis menyesuaikan tampilan layar, baik diakses melalui *smartphone*, tablet, maupun desktop.
*   **Meta Ads Ready:** Struktur kode yang bersih dan siap diintegrasikan dengan Meta Pixel untuk pelacakan konversi (*conversion tracking*).
